# Advanced To-Do App

A modern and responsive task management application built using HTML, CSS, and JavaScript.

## Live Demo
https://dikshachopra2007.github.io/Advanced-Todo-app/

## Features
- Add and delete tasks
- Mark tasks as completed
- Progress tracking
- Local storage persistence
- Responsive design
- Interactive UI

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Local Storage API
