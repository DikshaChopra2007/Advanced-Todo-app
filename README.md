
# 📝 Advanced To-Do App

A feature-rich and responsive task management application built with HTML, CSS, and JavaScript, featuring progress tracking, local storage persistence, and multiple customizable themes.

## Live Demo
https://dikshachopra2007.github.io/Advanced-Todo-app/

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Progress tracking system
- Persistent storage using Local Storage
- Multiple customizable themes and backgrounds
- Responsive design for desktop, tablet, and mobile devices
- Interactive animations and smooth user experience
- Clean and modern UI
## Tech Stack
- HTML5
- CSS3
- JavaScript
- Local Storage API
